# Advanced RAG Certificate -- Study Notes

**Course:** Building and Evaluating Advanced RAG Applications
**Provider:** DeepLearning.AI (Coursera)
**Completed:** 2024
**Instructors:** Jerry Liu (LlamaIndex) & Anyscale team

## Key Topics
1. Sentence-Window Retrieval -- decouple retrieval unit from context unit
2. Auto-Merging Retrieval -- hierarchical chunking with automatic promotion
3. RAG Evaluation Triad -- context relevance, groundedness, answer relevance
4. Advanced Chunking Strategies -- semantic, document-structure-aware, proposition-based
5. Reranking Approaches -- cross-encoder, LLM-based, Cohere Rerank

- Techniques applied to the Mental Health RAG System (Project 2)

- Chunking strategies tested empirically (200/300/500/800 token comparison)
