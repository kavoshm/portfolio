# Clinical Note Classifier

![Python](https://img.shields.io/badge/Python-3.11+-blue?logo=python&logoColor=white)
![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4-412991?logo=openai&logoColor=white)
![LangChain](https://img.shields.io/badge/LangChain-0.1+-green?logo=chainlink&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-yellow)

An LLM-powered classification system that categorizes unstructured clinical notes into structured urgency levels, department routings, and ICD-10 code suggestions. Built with prompt engineering techniques -- few-shot learning, chain-of-thought reasoning, and structured output formatting.

**Key Results:** 88% urgency accuracy, 92% department accuracy, 100% risk flag recall on 50 synthetic clinical notes.

**Key Tech:** Few-shot prompting, chain-of-thought reasoning, structured JSON output, GPT-4 (temp=0)
