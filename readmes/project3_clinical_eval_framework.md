# Clinical Evaluation Framework (Flagship Project)

![Python](https://img.shields.io/badge/Python-3.11+-blue?logo=python&logoColor=white)
![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4-412991?logo=openai&logoColor=white)
![Pytest](https://img.shields.io/badge/Pytest-Tested-green?logo=pytest&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-yellow)

A production-grade evaluation framework for scoring LLM-generated clinical outputs using G-Eval (reference-free LLM-as-judge) with four domain-specific rubrics developed in collaboration with clinical staff.

**Key Results:** Accuracy 4.18/5.0, Safety 4.72/5.0, zero safety hard-fails, ~$8.50 per full evaluation run

**Key Tech:** G-Eval, GPT-4-as-judge, probability-weighted scoring, baseline management, CI/CD integration
