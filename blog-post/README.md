## Why This Topic

Evaluation is the most undervalued skill in AI engineering. Most teams ship LLM features
with no systematic quality measurement. In healthcare, this is not just bad engineering --
it is a safety problem. This post shares concrete, hard-won lessons from building an
evaluation framework for clinical LLM outputs, targeting the gap between "we should evaluate
our AI" and "here is how to actually do it."
